🏃 ADL — Barthel Index

วัดความสามารถในการทำกิจวัตรประจำวัน

👤 ข้อมูลผู้สูงอายุ

📝 ADL 10 ข้อ (คะแนนเต็ม 20)

💭 2Q / 9Q ซึมเศร้า

คัดกรองภาวะซึมเศร้า

👤 ข้อมูลผู้รับการประเมิน

📝 2Q — คัดกรองเบื้องต้น

ใน 2 สัปดาห์ที่ผ่านมา รวมถึงวันนี้ มีอาการต่อไปนี้หรือไม่

1. รู้สึกหดหู่ ซึมเศร้า หรือท้อแท้สิ้นหวัง
2. รู้สึกเบื่อ ทำอะไรก็ไม่เพลิดเพลิน

📝 9Q — ระดับความรุนแรง

0=ไม่มีเลย   1=มีบางวัน   2=มีบ่อย   3=มีแทบทุกวัน

👤 ข้อมูลผู้สูงอายุ

0=ไม่เลย   1=นานๆครั้ง   2=บางครั้ง   3=บ่อย   4=เกือบตลอดเวลา

⚙️ ตั้งค่าฐานข้อมูล

เชื่อมต่อ Cloud เพื่อแชร์ข้อมูลระหว่างเครื่อง

📖 วิธีตั้งค่า Firebase (ทำครั้งเดียว ~3 นาที)

1. เปิด console.firebase.google.com → Sign in ด้วย Google
2. กด Create a project → ตั้งชื่อ → กด Continue จนเสร็จ
3. เลือก Realtime Database → กด Create Database
4. เลือก Start in test mode → กด Enable
5. คัดลอก URL ที่ขึ้นต้นด้วย https://xxx-default-rtdb.firebaseio.com
6. นำมาใส่ในช่องด้านล่าง → กด บันทึก ✅

🔗 ใส่ Firebase URL
